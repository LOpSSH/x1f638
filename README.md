# Setup
Create `gradle.properties` in the root project folder with content:

```
JAVA6_HOME=<PATH_TO_JDK_6>
JAVA7_HOME=<PATH_TO_JDK_7>
JAVA8_HOME=<PATH_TO_JDK_8>
```
